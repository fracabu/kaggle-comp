## Percorso Consigliato per fcWebDev

Basato sul tuo profilo (Frontend Developer -> Data Analyst), ecco un percorso ottimale:

### Fase 1: Fondamenta (Completare prima)

2. Pandas
3. Intro to SQL

### Fase 2: Data Analysis Core
4. Data Visualization
5. Data Cleaning
6. Advanced SQL

### Fase 3: Machine Learning
7. Intro to Machine Learning
8. Intermediate Machine Learning
9. Feature Engineering

### Fase 4: Specializzazione
10. Time Series (utile per analisi finanziarie)
11. Machine Learning Explainability

### Fase 5: Deep Learning (Opzionale)
12. Intro to Deep Learning
13. Computer Vision
