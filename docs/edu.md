## Workflow per Educatori

### Step 1: Crea Gruppo Classe

```
1. Vai a kaggle.com/groups
2. Click "New Group"
3. Nomina il gruppo (es. "Data Science 101 - Fall 2024")
4. Invita studenti via link o username
```

### Step 2: Prepara Contenuti

**Opzione A: Usa Risorse Esistenti**
- Cerca dataset pubblici
- Fork notebook esistenti
- Usa modelli pre-trainati

**Opzione B: Carica Materiali Custom**
- Upload dataset personalizzati
- Crea notebook da zero
- Configura competizioni specifiche

### Step 3: Condividi con la Classe

```
1. Vai alla risorsa (notebook/dataset)
2. Settings → Collaborators
3. Cerca nome del gruppo
4. Scegli permessi (Can View / Can Edit)
```

### Step 4: Crea Competition (Opzionale)

Per assignment valutati automaticamente:

```
1. +Create → Competition
2. Configura:
   - Dataset di training
   - Dataset di test (con labels nascosti)
   - Metrica di valutazione
   - Deadline
3. Condividi con gruppo classe
```

## FAQ

### Perché usare Kaggle e Colab?

- **Accesso gratuito** a compute (CPU/GPU/TPU)
- **Nessun setup** richiesto agli studenti
- **Riproducibilità** garantita
- **Real-world data** disponibili
- **Community** per supporto

### Come usare Notebook per insegnare?

1. Crea notebook con spiegazioni in Markdown
2. Aggiungi celle di codice con esercizi
3. Condividi con studenti
4. Studenti forkano e completano

### Come usare Dataset nel curriculum?

1. Cerca dataset rilevanti su Kaggle
2. Oppure carica dataset custom
3. Crea notebook di analisi
4. Assegna come homework

### Come creare Competition per classe?

1. Prepara dataset con train/test split
2. Definisci metrica di valutazione
3. Imposta deadline
4. Limita partecipazione al gruppo classe

### Come usare Colab per lezioni interattive?

1. Crea notebook con contenuto della lezione
2. Includi celle eseguibili
3. Studenti seguono live
4. Modificano e sperimentano

### Come condividere notebook con studenti?

1. Crea notebook
2. Click "Share"
3. Aggiungi gruppo o singoli utenti
4. Studenti ricevono notifica

### Risorse per iniziare?

- [Kaggle Learn Courses](https://www.kaggle.com/learn) - Corsi gratuiti
- [Documentation](https://www.kaggle.com/docs) - Guide tecniche

## Use Case per Team AI

1. **Onboarding** nuovi membri del team

3. **Practice competitions** interne

```
1. Crea gruppo "AI Team Training"
2. Condividi notebook di best practices
3. Crea mini-competition interne

5. Review e feedback sui notebook
```
